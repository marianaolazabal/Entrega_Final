Blog sobre Python

Este proyecto fue desarrollado utilizando Anaconda, ya que en mi computadora del trabajo no tengo permisos de administradora y no podía realizar las configuraciones necesarias directamente.

Página principal

La página index muestra el título “Blog sobre Python” y un botón “Ingresar”, que redirige al listado de publicaciones.

Listado de publicaciones

En esta sección se pueden:

Buscar publicaciones por letra o texto.

Crear nuevas publicaciones (solo disponible para usuarios autenticados).

Acceder al detalle de cada publicación mediante el botón “Leer”.

Creación de publicaciones

El formulario de creación solicita los siguientes datos:

Título

Contenido

Estado

Categoría

Para poder crear una publicación, es necesario iniciar sesión.

Existe un usuario administrador con las siguientes credenciales:

Usuario: admin

Contraseña: 123

El usuario administrador puede agregar y eliminar publicaciones.

Funcionalidad de los botones

El botón “Enviar” guarda la nueva publicación en la base de datos y redirige al listado.

El botón “Cancelar” simplemente vuelve al listado sin guardar cambios.

Detalle de publicación

Desde el listado, al hacer clic en “Leer”, se accede a la página de detalle de la publicación, donde se muestra toda la información completa.
